# 1.vue是 MVVM模型

是一种软件设计模式， 它是Model-View-ViewModel的缩写 （数据模型-视图-视图模型）

// 1. 创建一个Vue实例 
    // MVVM ==>
    // M:Model 代表数据模型，数据和业务逻辑都在 Model 层中定义。（对应data中的数据和业务逻辑）
    // V:View 代表UI视图,也就是用户界面，负责数据的展示。 （对应着template模板，负责数据的展示）
    // VM:viewModel 也叫做视图数据连接层, 负责监听 Model 中数据的改变并且控制视图的更新，处理用户交互操作, （对应着我们的Vue实例）

# 7.事件修饰符

https://v2.cn.vuejs.org/v2/guide/events.html#按键修饰符

 <div id="app">
    <div class="father" @click="fatherFn">
      <h1>vue获取事件对象</h1>
      <a href="http://baidu.com" @click.prevent.stop="gobai">去百度</a>
      <a href="http://jd.com" @click.stop.prevent="gojd('京东','猫粮', $event)">去京东</a>
    </div>
  </div>
  <script src="./js/vue.js"></script>
  <script>
    // 事件修饰符
    // 1. @事件名.prevent  ==> e.preventDefault() 阻止默认行为
    // 2. @事件名.stop     ==> e.stopPropagation() 阻止事件冒泡
    // 3. 事件修饰符可以串联 ==> @click.prevent.stop / @click.stop.prevent  没有顺序
    const vm = new Vue({
      el: "#app",
      methods: {
        gobai() {
          console.log('百度')

        },
        gojd() {
          console.log('京东')

        },
        fatherFn() {
          console.log('我是父盒子,事件冒泡了')

        }
      }
    })

# 10.vue局部组件

  <div id="app">
    <!-- 全局组件的复用 -->
    <button-counter></button-counter>
    <!-- 使用局部组件 -->
    <comp-a></comp-a>
  </div>
  <script src="./js/vue.js"></script>
  <script>

    // 定义一个名为 button-counter 的新组件
    // 1. 全局注册的组件 
    //    ==> 一个组件一旦全局注册了，我们可以在所有的组件中使用它
    Vue.component('button-counter', {
      data() {
        return {
          count: 0
        }
      },
      template: '<button @click="count++">You clicked me {{ count }} times.</button>'
    })

    // 2. 局部注册组件
    // 通过一个普通的JavaScript对象来定义组件
    // 我们可以定义很多很多局部组件，要使用的时候，就在components这个对象去注册
    const ComponentA = {
      data() {
        return {
          count: 0
        }
      },
      // 在局部注册的组件里，可以使用全局组件
      template: `
                <div>
                   <button-counter></button-counter>
                    <button @click="count++">我是局部组件{{ count }}</button>
                </div>
            `
    }

    const vm = new Vue({
      el: '#app',
      data() {
        return {
          msg: 'Hello World'
        }
      },
      components: {
        // 左边的是组件的名字， 在html上使用的时候
        // <comp-a></comp-a>
        'comp-a': ComponentA
      }
    })

# 12.脚手架cli

什么是脚手架

Vue官方提供的一个全局命令工具，可以快速创建一个Vue项目的基础架子
好处：
1.开箱即用 2.零配置(不用自己配置webpack) 3.内置babel等工具  babel是帮助解析其他语言的
1
2
3

安装

1.全局安装 
npm i @vue/cli -g   或 yarn global add @vue/cli
2.查看vue版本
vue --version
3.找一个目录，初始化一个vue项目(不能使用中文) vue create
4.cd 进入目录 ，启动项目，打包项目
启动：yarn serve 或 npm run serve
打包：yarn build 或 npm run build

# 17.sync修饰符

vue 修饰符sync的功能是：当一个子组件改变了一个 prop 的值时，这个变化也会同步到父组件中所绑定
//示例 
在父组件中
<addTop :isShow.sync="isShow"></addTop>   ///给子组件传值
export default{
data(){
    return {
        isShow:true
}
}}
子组件中
有一个方法需要改变isShow
export default{
props:{
isShow:{   //子组件接收值
    type:Boolean
}
},
 methods:{
  clone(){
  this.$emit('update:isshow',false)  //当我们需要在子组件修改父组件传过来的数据时，提交事件，加了sync修饰符的话，父组件那边就不用监听事件，可以直接修改数据了
  }
 }
}
