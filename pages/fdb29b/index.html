# 3.全局配置

app.json //全局配置   https://developers.weixin.qq.com/miniprogram/dev/reference/configuration/app.html
pages字段：用于描述当前小程序所有的页面路径
window: 用于设置小程序的状态栏、导航条、标题、窗口背景色https://developers.weixin.qq.com/miniprogram/dev/reference/configuration/app.html#window
常用的属性配置 
 "window": {
    "backgroundTextStyle": "dark", //设置下拉时loading时的样式
    "navigationBarBackgroundColor": "#3194ff",  //设置导航栏顶部的背景颜色
    "navigationBarTitleText": "黑马头条",   //设置导航栏标题的内容
    "navigationBarTextStyle": "white",   //设置内容的颜色
    "enablePullDownRefresh": true,  //支持下拉刷新
    "backgroundColor": "#efefef"   //下拉的背景色
  },
tabBar底部配置 其于配置自己去看文档吧，很简单
  "tabBar": {
    "list": [ //配置底部导航栏的名称和路径
      {
        "text": "首页",
        "pagePath": "pages/index/index"
      },
      {
        "text": "我的",
        "pagePath": "pages/my/my"
      }
    ]
  }

# 10.小程序的第三方依赖的使用

# 小程序第三方使用
- 注意事项
(1) 小程序只能使用部分npm包,涉及到dom和bom操作的npm包是不能使用的
(2) 通常去npm查找小程序可以用的包,加上`mp`或者`wx`前缀,例如`mp axios`,或者`wx axios`
- 步骤
  - 使用简单依赖,例如dayjs
    (1)生成package.json
      npm init -y
    (2)安装依赖
      npm i dayjs
    (3)点击菜单`工具>构建npm`
    (4)测试dayjs的使用
    在app.js

import dayjs from 'dayjs'
console.log(dayjs().format('YYYY'))
  **注意**每次安装新的依赖都需要`构建npm`
卸载依赖 在node_moudel文件下打开内置终端 ，输入命令  npm uninstall vant-weapp //卸载依赖，再构建一下npm
使用ui库vant小程序版本
(1)打开文档
  https://vant-contrib.gitee.io/vant-weapp/#/home
(2)安装依赖
  npm i @vant/weapp
(3)点击菜单`工具>构建npm`
(4)将 app.json 中的 "style": "v2" 去除
(5)全局注册组件
在app.json当中
 "usingComponents": {
    "van-button": "@vant/weapp/button/index"
  }

# 11.系统设置

点击菜单`设置`
    外观主题选择`深色`
    代理设置,选择`不使用任何代理`
(3)安装扩展
  找vscode的扩展C:\Users\用户名\.vscode\extensions(课程当中老师已经准备4个扩展为例子)
  点击菜单`工具>打开编辑器扩展目录`
  粘贴/解压扩展文件夹到小程序开发者工具里面去
  **注意**:安装扩展,需要重启小程序开发工具
(4)配置自动格式等操作
  输入ctrl+shift+p,打开命令面板
  搜索setting.json点击`打开首先项json`
  粘贴下面配置
  
  // js格式化使用的扩展    
"[javascript]": {
    "editor.defaultFormatter": "vscode.typescript-language-features"
},
// json格式化使用的扩展        
"[json]": {
     "editor.defaultFormatter": "vscode.json-language-features"
},    
//自动格式化  
"editor.formatOnSave": true,
//自动补充右花括号
"editor.autoClosingBrackets": "always",
//px转rpx    
"cssrem.wxss": true,

# 15.发请求

- 注意事项
  (1) 小程序没有跨域问题
  (2) 小程序发请求要求协议是https协议
    请求路径需要配置到微信管理后台
    一下特殊路径不能被配置
- 步骤
  (1)允许请求访问
  方式1-不校验路径(开发阶段)
  小程序开发工具,点击工具栏`详情>本地设置`,勾选`不校验合法域名`
  方式2-在微信开发管理后台配置路径(上线阶段)
  登录管理后台https://mp.weixin.qq.com/wxamp/index/index?lang=zh_CN&token=785907001
  点击左侧菜单,`开发>开发管理`,点击右侧`开发设置>服务器域名>修改`
  添加request合法域名`https://www.escook.cn;`
  (2)写代码请求数据
