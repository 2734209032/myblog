# 3.JSX原理

jsx本质上是js对象，是js的语法扩展，  jsx ---->(jsx转换器(CRA提供))-----> js对象 
是React.createElement的语法糖
1.//不使用jsx语法，创建元素
import React from 'react'
const h2 = React.createElement('h2',{className:'h2'},'我是嵌套的jsx');
const div = React.createElement('div',null,h2);
//渲染根对象
root.render(div);
2.使用jsx语法
作用：写 React 页面结构，类似 HTML
规则：① 唯一根节点 ② 标签必须闭合 ③ 属性名称小驼峰命名法
//如果不想根节点 加标签，可以用幽灵标签
可以用<> </> 或<React.Fragment>  </React.Fragment>  //不占有位置，但有标签的作用
 //标签必须闭合
 eag:  const ele3 = <input> </input>  或 <input/>
 //属性名称小驼峰命名法
backgroundColor
 class要用calssName代替，style要用双大括号,{ {} } --最外层是语法要求，里面是对象，
eag：
const ele4 =<div className='' style ={{ fontSize:12}}></div>

# 5.JSX中{ } 的作用

<div> {javaScript 表达式} </div>
作用：让jsx变为动态的，可以用在标签内容，属性中
常见应用场景： 1. 列表渲染 2. 条件渲染 3.样式处理
1.表达式 ?? //逻辑空运算符
const eal = undefined / null
空值合并运算符（??）是一个逻辑运算符，当左侧的操作数为 null 或者 undefined 时，返回其右侧操作数，否则返回左侧操作数。
const foo = null ?? 'default string';
console.log(foo);
// Expected output: "default string"
const baz = 0 ?? 42;
console.log(baz);
// Expected output: 0
2.? 
null?.name  //如果前面不是null或unfined 才会执行后面的.name
3.~~ 取整运算符
~~3.333 //3

# 7.react组件

React 组件是一个 JavaScript 函数，函数中可以添加 JSX 标记
//chilend.jxs中
创建组件
function Childen (){
    return(
    <div>我是子组件</div>
    )
}
//导出组件
export default Childen
//在App.jsx中
import Childen from './Childen'
const App =()=>{
    return (
    <div>我是App组件</div>
        //使用子组件
      <Childen></Childen>
    )
}
注意点：
1.React 组件是常规的 JavaScript 函数，但它们的名称必须以大写字母开头，否则它们将无法正常工作！
2.如果没有括号，后面行中的任何代码都将被忽略！return
3.<Childen />以大写字母开头，所以 React 知道我们要使用名为 的组件
4.当子组件需要来自父组件的某些数据时，请通过 props 而不是嵌套定义传递它。

# 8.组件的状态

状态（state）是可以让页面内容发生变化的数据
使用状态: useState 函数
eag: 
import {useState}
const [list,setList] = useState([])
//useState 函数返回一个数组，第一个数组参数是，我们定义的变量，第二个参数是我们需要修改变量的函数，它的原理是闭包
注意点:
1.useState只能用在函数式组件或者是自定义hook中
2.不能放在if else 也不能for循环，必须至于组件的顶级
