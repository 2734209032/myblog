es6模块化补充

# 1. 模块化 ESM

Node.js ===> CommonJS规范 默认支持 ===> package.json ==>配置 type:"module"

浏览器要使用ESM ==> script type="module"

# 2. 导入导出

1. 默认导出   export default 
2. 按需导出   export 变量名
3. 默认导入   import xxx from '路径.js'   ==> .js后缀不能省略，不能是模板字符串
4. 按需导入   import { 变量名 } from '路径.js'
5. 全部导入   import * as 变量名 from '路径.js' 
6. 直接导入   import '路径.js'  ==> 直接执行.js中的代码
