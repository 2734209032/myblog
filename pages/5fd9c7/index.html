# 1.this是什么

执行上下文中包含了变量环境、词法环境、外部环境，但其实还有一个 this 
对应的 this 也只有这三种——全局执行上下文中的 this、函数中的 this 和 eval 中的 this。
1
2
1.全局执行上下文中的this环境是指向window的
2.函数执行上下文中的this指向的是调用者
3.对象中的方法调用 指向对象
4.默认调用函数指向window
5.定时器中this指向window
6.绑定事件中的回调函数，this指向事件源

在全局环境中调用一个函数，函数内部的 this 指向的是全局变量 window。
通过一个对象来调用其内部的一个方法，该方法的执行上下文中的 this 指向对象本身。 这里就能得出一个结论，函数中的this环境不是在定义的时候确定的，而是在使用的时候确定的，箭头函数并不会创建其自身的执行上下文，所以箭头函数中的 this 取决于它的外部函数。
